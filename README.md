## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/JustLukaBraza/portfolio.git
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

4. Open your browser and navigate to:
```
http://localhost:5000
```
